## Getting Started

Visit <https://alexjsully.me> to view the live portfolio.

To run locally:

```bash
git clone https://github.com/AlexJSully/alexjsully-portfolio.git

cd alexjsully-portfolio

npm ci

npm run dev
```

The app will be available at <http://localhost:3000/>.
